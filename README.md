# 🧮 GaussPlay

O **GaussPlay** é uma ferramenta interativa e educativa desenvolvida em Python para auxiliar no aprendizado de **Escalonamento de Sistemas Lineares** (Método de Eliminação de Gauss). Com uma interface moderna e intuitiva, o usuário pode realizar operações elementares em matrizes de forma manual ou automática, visualizando o passo a passo matemático.

---

## 🚀 Funcionalidades

*   **Configuração Dinâmica**: Suporte para sistemas de $2 \times 2$ até $5 \times 5$.
*   **Modo Aleatório ou Manual**: Gere desafios automáticos ou insira sua própria matriz para resolver.
*   **Operações Elementares**:
    *   Troca de linhas ($L_1 \leftrightarrow L_2$).
    *   Multiplicação de linha por escalar ($L_1 \to k \cdot L_1$).
    *   Combinação linear ($L_1 \to L_1 + k \cdot L_2$).
*   **Cálculos Precisos**: Uso da biblioteca `fractions` para manter a precisão exata com frações, evitando erros de arredondamento.
*   **Histórico de Ações**: Botões de **Desfazer** e **Refazer** para explorar diferentes caminhos de resolução.
*   **Interface Moderna**: Desenvolvido com `customtkinter`, oferecendo um visual limpo e suporte a temas.

---
